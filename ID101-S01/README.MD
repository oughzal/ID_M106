## Exercice 1
Écrire un programme qui demande un nombre à l’utilisateur, puis qui calcule et affiche le carré de ce nombre.
```python
N = float(input("Donner un nombre : "))
C = N**2
print(N,"^2 =" , C)
print("Le carré de ", N , " est ", C)
print("le carré de ", N, " égale à ", N*N)
```
